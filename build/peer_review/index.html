Peer reviewing assignments

This page should (:D) show you three assignments: one made by you, and two made by other people. If you didn't complete creating an assignment, you will be shown three assignments made by other people.

Peer review

Below you will find three assignments created in CrowdSorcerer: two by other students, and one for self-evaluation. The instructions the assignments are based on are included to refresh your memory.

Inspect all the parts of the exercise: assignment, code template, model solution, inputs and the outputs. Evaluate their clarity, difficulty and how well they match the given instructions.

You can change the view between code template and model solution by clicking the buttons above the source code. Some questions have been given to make review process easier. Choose how well the statement applies to the exercise by choosing smileys from the scale. You should also give some written feedback. Lastly, add some tags that you think describe the exercise best and hit "Send".

Give a review for both the exercises done by other users, as well as your own.

Remember to be fair and friendly. Good feedback is honest, but supportive!

The assignments can be downloaded as a ZIP-file that includes both the code template and the model solution. This way you can test the program on the IDE of your choice.
