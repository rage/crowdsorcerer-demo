To report an issue in this demo material

You can report mistakes and mishaps found to the Issues-page of this material's GitHub repository, whether they concern the material itself or the tool. Before you create a new issue, please check that the problem hasn't already been reported in here.

Create a new issue

How to create a new issue

  1. Click the green button above.
  2. Give your issue a nice and concise title that describes the problem, e.g. "Typo in the setup page".
  3. Write longer description to the comment field, e.g. "In the first paragraph, the page says to use "the fool" when it should be "the tool"".
  4. To submit, click the green "Submit new issue" button.

We may ask defining questions about your issue if we are not able to fix the problem without your help. These questions should send you a notification email to the email address connected to your GitHub account.
